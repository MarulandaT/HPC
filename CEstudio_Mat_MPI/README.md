# Para ejecutar los algoritmos Seriales y paralelo con MPI en el clúster: 

Con el fichero matMPI.c montado en la carpeta del NFS del cluster ejecutar el comando: 
"bash execute.sh compile run matMPI"

Este comando generará un archivo de resultados "times" con la relación entre: 
#procesos (worker nodes);#Dimensión de la Matriz;#TiempoEjecución[s]

Ordenada como una tabla con 10 intentos de cada dimensión de matriz.
